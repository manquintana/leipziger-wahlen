# Leipzig Mayoral Elections Analysis (1994–Present)

This project explores the evolution of vote shares per party in Leipzig mayoral elections since 1994. The focus is on the **first round results** of each election, analyzing participation rates and vote distribution over time.

## Data Source

The dataset is based on official election data: **Oberbürgermeisterwahlen – 1. Wahlgang** from Leizig Open Data Portal (https://opendata.leipzig.de)
> Results of the first round of mayoral elections in Leipzig, including voter turnout and vote shares per candidate since 1994.


## Data Pipeline

### 1. Data Retrieval & Preprocessing (Python)

- Data is collected from the official source, cleaned and standardized using Python

### 2. Data Visualization (Google Data Studio)

- The processed dataset is imported into **Google Data Studio** where the dashboards are created and hosted
